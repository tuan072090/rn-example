#  The base setup for react native and firebase

##  Included
- React navigation
- Firebase analytics
- Firebase authen
- Firebase cloud messaging
- Firebase crashlytics
- Firebase dynamic links


##  How to run
- pull the project, then `yarn install`, then `cd ios && pod install`, then `yarn ios`
